## Client Customization Checklist

Visit [http://localhost:7700/dev/todo](http://localhost:7700/dev/todo) (development only) to see a comprehensive checklist of customization tasks for this client project.

The checklist includes:
- 🎨 Branding & Design (colors, fonts, logos)
- 📝 Content & Metadata (titles, descriptions, copy)
- 🖼️ Assets & Media (favicons, OG images)
- ⚙️ Configuration (env vars, API setup)
- ✅ Quality & Testing (accessibility, performance)
- 🚀 Deployment (hosting, domain, SSL)

Progress is saved in `.dev-todo.json` and shared with the team.
